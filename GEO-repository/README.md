## GEO / SEO лендинг

Этот репозиторий содержит одностраничный сайт на русском языке для продвижения GEO / SEO-оптимизации, по структуре и смыслу похожий на сайт компании [Next Net](https://nextnet.ai/), но адаптированный под локальный рынок.

### Запуск

Откройте файл `index.html` в браузере (двойной клик по файлу или через контекстное меню «Open With…»).

### Стэк

- Чистый HTML5
- CSS3 (адаптивная вёрстка)
- Небольшой JS для интерактивности
